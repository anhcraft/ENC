# ENC
Brings more enchantments to your servers

### Maven Integration
```text
<repositories>
    <repository>
        <id>anhcraft-repo</id>
        <url>https://repo.anhcraft.dev/repository/mvn/</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>dev.anhcraft</groupId>
        <artifactId>enc</artifactId>
        <version>VERSION</version>
        <scope>provided</scope>
    </dependency>
</dependencies>
```
